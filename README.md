# VueBoard

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### 안되면~
```
rm -rf node_modules
```
->
```
npm install
```
->
```
npm run serve
```
